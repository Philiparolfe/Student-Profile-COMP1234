Lists:


List 1

  1. type = ”1” The list items will be numbered with numbers (default)
  2. type = ”a” The list items will be numbered with lowercase letters
  3. type = ”A” The list items will be numbered with uppercase letters
  4. type = ”i” The list items will be numbered with lowercase roman numbers
  5. type = “I” The list items will be numbered with uppercase roman numbers


List 2

  • "list-style-type: disc;" Sets the list item marker to a bullet (default)
  • "list-style-type: circle;" Sets the list item marker to a circle
  • "list-style-type: square;" Sets the list item marker to a square
